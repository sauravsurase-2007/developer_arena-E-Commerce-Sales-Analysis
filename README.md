# 📊 E-Commerce Sales Analysis

## Project Description

This project analyzes an E-Commerce Sales dataset using Python.

The project performs:

- Data Loading
- Data Cleaning
- Data Analysis
- Data Visualization
- Report Generation

---

## Features

- Load CSV Dataset
- Clean Missing Values
- Total Sales
- Average Sales
- Highest Sale
- Lowest Sale
- Sales by Category
- Monthly Sales Analysis
- Bar Chart
- Line Chart
- Pie Chart

---

## Technologies Used

- Python
- Pandas
- Matplotlib
