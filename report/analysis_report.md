# 📊 E-commerce Sales Analysis Report

## Project Overview

This project analyzes an E-commerce Sales dataset using Python. The analysis includes data cleaning, sales calculations, and data visualization using Matplotlib.

---

# Dataset Information

- Dataset Name: sales_data.csv
- Total Records: 100
- Columns: 5
- Technologies Used:
  - Python
  - Pandas
  - Matplotlib

---

# Objectives

- Analyze sales performance.
- Find top-selling products.
- Compare sales by category.
- Study monthly sales trends.
- Visualize data using charts.

---

# Data Analysis

## Total Sales

Total sales were calculated by summing all sales values.

## Average Sales

Average sales were calculated to understand the overall sales performance.

## Highest Sales

The product with the highest sales was identified.

## Lowest Sales

The product with the lowest sales was identified.
